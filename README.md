# Meme Generator

A React meme generator built with Vite. Pick an image template, add top and bottom text (white with black outline), resize the text, and download the result.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Features

- **Template image** – Choose an image from your device
- **Top & bottom text** – Classic meme layout
- **Resizable text** – Slider from 12px to 120px
- **White text with black border** – Readable on any background
- **Download** – Export the meme as PNG

## Build

```bash
npm run build
npm run preview   # preview production build
```
